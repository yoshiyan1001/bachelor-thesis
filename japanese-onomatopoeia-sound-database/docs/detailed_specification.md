# Detailed specification
## Japanese Onomatopoeia Database - Project Analysis
### 1. Existing project Analysis
#### Challenges with Conventional Audio and Onomatopoeia Systems
There are main five challenges in existing systems.
- **Simple dictionary-style applications** <br>
There are several cases that existing onomatopoeia-related tools include only text-based dictionaries or static audio sample collections.
- **Lack of Machine Learning**<br>
There are few applications that have the function to predict onomatopoeic words dynamically from audio files.
- **Lack of analytical capabilities** <br>
 Few other applications contain analysis functions for researchers.
## 2. Key Differentiators from Existing Programs
### The Strengths of This Project
- **Audio file &rarr; automatic prediction of onomatopoeic word**<br>
This application has the prediction function based on machine learning, which is not included in existing dictionary-based applications.
- **Quantitative audio analysis**<br>
This application can compare quantified features of audio files.
- **Visualizing Relationships in 3D Space**<br>
This function can help users understand similarity of onomatopoeic words intuitively.
- **Practicality for Japanese language learners** <br>
Users can study onomatopoeic words from audio recordings.
